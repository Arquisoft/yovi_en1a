== Deployment View

=== Infrastructure Level 1: Azure Cloud Environment

The system is deployed on a single Azure Virtual Machine running Ubuntu. All components, including the frontend, are containerized to ensure consistency between development and production.

[plantuml, level1-whitebox, svg]
----

  }

    component [Webapp] as webapp <<Nginx/Static>>
    component [User Service] as usersvc <<Node.js>>
    component [GameY Engine] as gamey <<Rust>>
    component [Monitoring] as mon <<Prometheus/Grafana>>
  }
}

@enduml

The following diagram illustrates the internal monitor-net bridge network and how the containers interact. Note that the Webapp is served via a container on port 80, and Grafana maps internal port 3000 to external 9091.

=== Mapping of Building Blocks to Infrastructure

[cols="1,2,2"]
|===
|Building Block |Infrastructure Element |Description

|webapp
|Docker Container (Nginx)
|Serves the React/Vite frontend. Accessible via Port 80.

|users
|Docker Container (Node.js)
|Handles identity, history, and stats. Accessible via Port 3000.

|gamey
|Docker Container (Rust)
|Logic for Game Y moves and win validation. Accessible via Port 4000.

|prometheus
|Docker Container
|Scrapes metrics from the services via the internal network.

|grafana
|Docker Container
|Visualizes metrics. Mapped to Port 9091 to avoid conflict with User Service.
|===

=== Infrastructure Level 3: Data Persistence

|===
|Element |Storage Strategy |Configuration Path

|Metrics & Config
|Docker Bind Mount
|./users/monitoring/prometheus mapped to /etc/prometheus

|Dashboards
|Docker Bind Mount
|./users/monitoring/grafana/provisioning for automated setup.

|User Data
|Docker Volume
|Persistence for user profiles and match history (recommended to add a named volume).
|===

=== Network and Connectivity

- Port 80: Entry point for the web application.

- Port 3000: API endpoint for User/History management (used by Webapp and Bots).

- Port 4000: API endpoint for Game Engine moves (YEN notation).

- Port 9090/9091: Administrative access for system observability (Prometheus and Grafana respectively).
